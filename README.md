
  # Factors Influencing Life Expectancy

## Project Overview
This project analyzes the key factors that influence life expectancy across different countries using statistical and data analysis techniques.

## Dataset
The dataset includes variables such as:
- Life expectancy
- Adult mortality
- Infant deaths
- Alcohol consumption
- GDP
- Schooling
- Health expenditure

## Objective
To identify and understand how socio-economic and health-related factors impact life expectancy.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Conclusion
The analysis shows that higher schooling, better healthcare, and higher income levels are strongly associated with increased life expectancy.
