# Factors Influencing Life Expectancy

## Overview
This project studies the factors that influence life expectancy using real-world health and socio-economic data.

## Dataset Description
The dataset contains information such as:
- Life expectancy
- Adult mortality
- Infant deaths
- Alcohol consumption
- GDP
- Schooling
- Health expenditure

## Aim of the Project
To analyze how different health and economic factors affect life expectancy across countries.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Key Insights
- Higher education and income levels are associated with higher life expectancy.
- Improved healthcare reduces mortality rates.
- Developing countries generally have lower life expectancy compared to developed countries.

## Conclusion
Life expectancy is strongly influenced by education, healthcare, and economic conditions.
Improving these factors can significantly increase life expectancy.
